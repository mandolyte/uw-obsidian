## Senses: 


### Sense  1.0: 

#### Definition: 

used to describe something done in a luxurious way 

#### Glosses: 

splendidly, luxuriously 

#### Explanation: 

frequently used of sumptuous food (compare Sir 29:22). λαμπρῶς is used to describe something done in a luxurious way such as how a rich person does things
